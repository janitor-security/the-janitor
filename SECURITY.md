# Security Policy

## Reporting a Vulnerability

Report vulnerabilities to [security@thejanitor.app](mailto:security@thejanitor.app).

Include the affected version, reproduction steps, impact, and any supporting
artifacts needed to validate the issue deterministically.

## Supported Versions

We support the current major version of The Janitor with security fixes and
coordinated disclosure handling.

## Disclosure Expectations

Do not disclose a vulnerability publicly until remediation is available or a
coordinated disclosure window has been agreed in writing.
