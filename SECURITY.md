# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub Private Vulnerability
Reporting:

**https://github.com/janitor-security/the-janitor/security/advisories/new**

Do not open public issues for security flaws. All reports are triaged within
48 hours.

## Disclosure Policy

We follow coordinated disclosure. After a vulnerability is verified, we target
a fix within **90 days** before public disclosure. We will work with reporters
to agree on an appropriate timeline based on severity.

## Contact

If GitHub Private Vulnerability Reporting is unavailable, contact us directly:

**security@thejanitor.app**
